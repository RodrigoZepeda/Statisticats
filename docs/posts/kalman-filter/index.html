En esta entrada discutimos la idea que inspir√≥ al filtro de Kalman. Empezamos con m√≠nimos cuadrados, pasamos a m√≠nimos cuadrados recursivos y finalmente al planteamiento y soluci√≥n del filtro de Kalman cl√°sico.

M√≠nimos cuadrados

El problema cl√°sico de regresi√≥n consiste en estimar \(\mathbf{\beta} = (\beta_1, \beta_2, \dots, \beta_p)^{\intercal}\) dadas \(n\) observaciones \(\mathbf{x}_t = (x_{1,t}, \dots, x_{p,t})^{\intercal}\) (\(t = 1, \dots,n)\) y \(\mathbf{y}_t = (y_1, \dots, y_t)^{\intercal}\) las cuales est√°n relacionadas de la siguiente forma:

\[ y_t = \sum_{i = 1}^{p} x_{i,t}\beta_i + \epsilon_t = \mathbf{x}_t^{\intercal} \mathbf{\beta} + \epsilon_t \] donde los t√©rminos de error \(\{\epsilon_t\}_{t=1}^n\) son variables aleatorias independientes con media cero y varianza \(\sigma^2 < \infty\).

Una de las formas cl√°sicas de estimaci√≥n es minimizando la suma de cuadrados de los errores de estimaci√≥n dada por:

\[ S(\mathbf{\beta}) = \sum_{t=1}^n \big(y_t - \mathbf{x}_t^{\intercal}\mathbf{\beta} \big)^2 \] La cual puede escribirse de manera matricial como la minimizaci√≥n de la norma-2: \[ S(\beta) = (\mathbf{y} - \mathbf{X} \mathbf{\beta})^{\intercal}(\mathbf{y} - \mathbf{X} \mathbf{\beta}) = \|\mathbf{y} - \mathbf{X} \mathbf{\beta}\|^2_2 \tag{1}\]

con \(\mathbf{X} = \big(\mathbf{x}_1|\mathbf{x}_2|\dots|\mathbf{x}_n\big)^{\intercal}\) y \(\mathbf{\epsilon} = (\epsilon_1, \dots, \epsilon_n)^{\intercal}\).

Si la matriz \(\mathbf{X}\) es de rango completo, la soluci√≥n al sistema es la proyecci√≥n ortogonal:

\[ \hat\beta=(\mathbf{X}^{\intercal}\mathbf{X})^{-1}\mathbf{X}^{\intercal}y = \left(\sum\limits_{t=1}^n \mathbf{x}_t \mathbf{x}_t^{\intercal} \right)^{-1} \sum_{t=1}^n\mathbf{x}_t \mathbf{y}_t \]

Y el resultado es la l√≠nea (hiperplano) que minimiza los errores:

C√≥digo 
import matplotlib.pyplot as plt
import numpy as np
import statsmodels.api as sm
import seaborn as sns
sns.set(rc={'text.usetex' : True})
sns.set_theme(style="white")

#Simulamos datos
np.random.seed(28802574)
x       = np.linspace(0,10)
epsilon = np.random.normal(loc=0.0, scale=2.5, size=len(x))
y_true  = 3*x + 15
y_obs   = y_true + epsilon

#Agregamos una columna de 1's a las X para el intercepto
x_extended = sm.add_constant(x)

#Obtenemos modelo de regresi√≥n
model = sm.OLS(y_obs, x_extended)

#Ajustamos modelo
results = model.fit()

#Predecimos
y_pred = results.predict()

#Gr√°fica
f, ax = plt.subplots(figsize=(8, 4.5))
f.suptitle('Estimaci√≥n por m√≠nimos cuadrados')
sns.despine(f, left=True, bottom=True)
sns.scatterplot(x=x, y=y_obs, color="#497174")
sns.lineplot(x=x, y=y_pred, color='#EB6440', alpha=0.5)
ax.set_title(r'$y = \beta_0 + \beta_1 x$')
ax.set_xlabel(r'$x$')
ax.set_ylabel(r'$y$')
plt.show()

M√≠nimos cuadrados recursivos

Si imaginamos un escenario donde los datos llegan de manera ordenada (como en una serie de tiempo) \(\mathbf{x}_1, \mathbf{x}_2, \dots\) podemos pensar en generar distintos estimadores de \(\beta\) dados por la colecci√≥n \(\{\hat{\beta}_i\}_{i=1}^n\) generados conforme se observan m√°s \(\mathbf{x}\). Es decir, el primer estimador de \(\beta\), \(\hat{\beta}_1\), se estimar√≠a s√≥lo con \(y_1\) y \(\mathbf{x}_1\) mientras que un estimador al tiempo \(t\), \(\hat{\beta}_t\), se generar√≠a con las primeras \(t\) observaciones: \(y_1, \dots, y_t\) y \(\mathbf{x}_1, \dots, \mathbf{x}_t\).

Las betas se pueden estimar recursivamente pues su diferencia es una matriz que s√≥lo depende del t√©rmino m√°s reciente (ver Triantafyllopoulos (2021) para su obtenci√≥n) como:

\[ \hat\beta_t = \hat\beta_{t-1} + K_t e_t \] donde \(K_t = \Big(\sum_{i=0}^{t-1} \mathbf{x}_{t-i} \mathbf{x}_{t-1}^{\intercal}\Big)^{-1} \mathbf{x}_t\) y \(e_t = (\mathbf{y}_t - \mathbf{x}_t^{\intercal} \hat{\mathbf{\beta}}_{t-1})\).

Podemos generalizar √©sta idea de estimaci√≥n secuencial decidiendo agregar un factor de descuento \(\delta\in(0,1]\). De esta manera para la suma de cuadrados pesar√≠a m√°s la informaci√≥n reciente:

\[ S(\mathbf{\beta}) = \sum_{t=1}^n \delta^{n-i}\big(y_t - \mathbf{x}_t^{\intercal}\mathbf{\beta} \big)^2 \]

√âsta misma suma se puede reescribir como una minimizaci√≥n cl√°sica de cuadrados considerando \(\tilde{\mathbf{X}}_t= \big(\delta^{(t-1)/2}\mathbf{x}_1|\delta^{(t-2)/2}\mathbf{x}_2|\dots|\delta^{0}\mathbf{x}_t\big)^{\intercal}\) y \(\tilde{\mathbf{y}}_t = (\delta^{(t-1)/2}y_1, \dots, \delta^{0}y_t)^{\intercal}\) en la ecuaci√≥n 1. El estimador \(\hat\beta_t\) de \(\beta\) al momento \(t\) est√° dado por:

\[ \hat{\beta}_t = (\tilde{\mathbf{X}_t}^{\intercal}\tilde{\mathbf{X}_t})^{-1}\tilde{\mathbf{X}_t}^{\intercal}\tilde{y}_t \] En el siguiente gr√°fico podemos ver c√≥mo var√≠a la estimaci√≥n de \(\beta\) dada por \(\hat{\beta}_t\) conforme avanza \(t\):

#Obtenemos modelo de regresi√≥n

#Ajustamos modelo
results = model.fit()

#Predecimos
y_pred = results.predict()

#Obtenemos los coeficientes
beta0 = results.recursive_coefficients.filtered[0]
beta1 = results.recursive_coefficients.filtered[1]
time  = range(0, len(x))

#Gr√°fica
f, ax = plt.subplots(1,3, figsize=(8,4.5))
f.suptitle('Estimaci√≥n por m√≠nimos cuadrados recursivos')

sns.lineplot(ax=ax[0], x=time, y=beta0, color='#497174', alpha=0.5)
ax[0].set_title(r'Intercepto ($\beta_0$)')
ax[0].set_xlabel(r'$t$')
ax[0].set_ylabel(r'$\beta_0$')

sns.lineplot(ax=ax[1], x=time, y=beta1, color='#EB6440', alpha=0.5)
ax[1].set_title(r'Pendiente ($\beta_1$)')
ax[1].set_xlabel(r'$t$')
ax[1].set_ylabel(r'$\beta_1$')

sns.scatterplot(ax=ax[2], x=x, y=y_obs, color="#497174")

ax[2].set_title(r'$y = \beta_0 + \beta_1 x$')
ax[2].set_xlabel(r'$x$')
ax[2].set_ylabel(r'$y$')
